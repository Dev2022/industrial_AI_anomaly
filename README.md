# Industrial AI + Predictive Maintenance Optimization Pipeline

End-to-end Industrial AI and Operations Research pipeline for predictive maintenance optimization using anomaly detection, machine learning, explainable AI, Remaining Useful Life (RUL) prediction, and MILP-based maintenance scheduling.

---

# Project Overview

This project simulates an industrial manufacturing environment where multiple machines continuously generate sensor telemetry data.

The system performs:

- Synthetic industrial sensor simulation
- Feature engineering on time-series data
- Anomaly detection using Isolation Forest
- Failure prediction using XGBoost
- Remaining Useful Life (RUL) prediction
- SHAP explainability analysis
- Maintenance optimization using MILP
- Cost reduction analysis
- CSV and visualization artifact generation

The project combines:

- Machine Learning
- Predictive Maintenance
- Explainable AI
- Operations Research
- Industrial Optimization
